# MuteComputers

Веб-приложение для проверки совместимости и сравнения характеристик комплектующих компьютера.

Работующее веб-приложение можно посмотреть по текущей [ссылке](https://8080-79c43b1233cf42f7a2da9ca5ab19d2d7.onpatr.cloud/).
